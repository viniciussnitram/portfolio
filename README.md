# Vinícius Martins - Portfolio

## Descrição

Projeto FrontEnd do curso Origamid. É um Portfolio inicial 
mostrando um pouco sobre experiências, formações e contatos.

## Objetivo do Projeto

Aprimorar o HTML e CSS. 
Entender e utilizar conceitos como: 
- HTML Semântico;
- CSS Seletores;
- Box Model;
- Display;
- CSS Posicionamento;
- CSS Propriedades;
- Responsivo.
